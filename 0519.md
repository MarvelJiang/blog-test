# 正则表达式

## 入门
1. 精准匹配一个单词(如hi): \bhi\b
2. 精准匹配以hi开头，hello结尾的句子(不能换行): \bhi\b.*\bhello\b
3. 精准匹配所有区号3位的电话号码:0\d\d-\d\d\d\d\d\d\d\d || 0\d{2}-\d{8}

## 其他元字符
1. \w用于匹配字母、数字、下划线、汉字:\b\w{6}\b找到所有6个字符的单词
2. \d+用于匹配1个或更多连续字符
3. ^: 匹配字符串的开始
4. $: 匹配字符串的结尾 ^\d{5,12}$ 匹配5位到12位字符的单词
5. 转义字符\

## 重复
1. *: 重复0次或更多次
2. +: 重复一次或更多次
3. ?: 重复零次或一次
4. {n}: 重复n次或更多次
5. {n,m}: 重复n到m次
